# Arduino Parking Sensor System

## Overview
This project measures distance using an ultrasonic sensor and provides alert using LED and buzzer.

## Components Used
- Arduino UNO
- Ultrasonic Sensor (HC-SR04)
- Buzzer
- RGB LED

## Working
The ultrasonic sensor measures distance and sends signal to Arduino. Based on distance, LED color changes and buzzer gives alert.

## Features
- Real-time distance measurement
- Accurate up to 4 meters
- Visual and audio alert system
